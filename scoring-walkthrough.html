8  Scoring Walkthrough

Interactive step-by-step visualization of the sensitivity scoring pipeline

This interactive walkthrough demonstrates how the Marine Sensitivity Toolkit calculates sensitivity scores, from input rasters through model merging, ecoregional rescaling, and final aggregation. Use the arrow keys or navigation buttons to step through the pipeline. Click any step in the top bar to jump directly to it.

For the underlying methodology, see Scoring and Model Merging. For source code, see the merge_models and calc_scores workflows.
